## Requirements

- Java 17 (64-bit)
- Minecraft 1.18.2 with Forge 40.2.0

## Build

```bash
.\build.bat
```

The jar is created in `build\libs\nightfall-1.18.2-1.0.0.jar`.

## Run the dev client

```bash
.\run-client.bat
```
